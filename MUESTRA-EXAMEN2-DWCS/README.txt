Propuesta de solución para cada pregunta de cada examen de cada unidad 05, 06, 07 (para la unidad 04 no hay propuesta, pues entiendo que es accesible)

Para cada examen de cada unidad didáctica, aparecen las propuestas de solución con el formato siguiente, por ejemplo para examen UNIDAD 05:

UNIDAD-05-00: Punto de partida
UNIDAD-05-02: Respuesta a la pregunta 2 del examen UNIDAD 05
UNIDAD-05-03: Añadida respuesta a pregunta 3 
y así sucesivamente

Usa la extensión "Compare folders" (de MoshFeu) para ver lo que se va haciendo de un paso a otro: 
  1) si no tienes instalada la extensión, la instalas
  2) seleccionas 2 carpetas que deseas comparar, por ejemplo UNIDAD-05-00 y UNIDAD-05-02
  3) en el menú contextual seleccionas [Compare folders] Compare selected folders
y miras qué archivos y/o carpetas fueron tocados, cuáles creados, etc.

Entiendo que así será más fácil entender qué se corresponde con cada pregunta.
